Dự án này là hệ thống Backend cho trang thương mại điện tử, sử dụng ASP.NET Core Web API và SQL Server.

## 🚀 Hướng dẫn khởi chạy nhanh

Dự án đã được cấu hình **Auto-Migration** và **Data Seeding**

1. **Cấu hình Database:**
   - Mở file `appsettings.json`.
   - Chỉnh sửa chuỗi `ConnectionStrings` sao cho khớp với SQL Server

2. **Chạy ứng dụng:**
   - Mở terminal tại thư mục dự án và gõ: dotnet run
